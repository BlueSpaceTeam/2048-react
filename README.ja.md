## 使用法

1. 開ける [http://2048.ued.team/](http://2048.ued.team/)。
2. `Play` をタッチすると、ゲームを開始し、`Ranking` をタッチすると、グローバルランキングが表示されます。
3. 使用`arrow keys`または、画面をスワイプしてタイルを移動します。
4. 同じ番号のタイルは、触れると1つに統合されます。
5. それらを合計して2048に到達します！
