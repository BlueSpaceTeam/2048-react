## 使用法

1.  開ける[ｈっｔｐ：／／２０４８。うえｄ。てあｍ／](http://2048.ued.team/)。
2.  接する`Play`ゲームを開始し、タッチします`Ranking` to view the global rankings.
3.  使用`arrow keys`または、画面をスワイプしてタイルを移動します。
4.  同じ番号のタイルは、触れると1つに統合されます。
5.  それらを合計して2048に到達します！
