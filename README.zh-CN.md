## 用法

1.  打开[HTTP://2048.UE的.team/](http://2048.ued.team/).
2.  触碰`Play`开始游戏，点击`Ranking`查看全球排名。
3.  使用`arrow keys`或滑动屏幕移动瓷砖。
4.  相同编号的瓷砖在接触时会合二为一。
5.  将它们加起来达到 2048！
